## Direktori
```.
├───.vscode
├───bin
├───doc
├───src
└───test
```
### Penjelasan Tiap Direktori
1. `bin`: menyimpan file executable 
2. `doc`: dokumentasi dan laporan akhir
3. `src`: menyimpan kode program
4. `test`: menyimpan kasus uji

## Kompilasi Program
pada /src, gunakan perintah
```bash
g++ source.cpp -o source
```

setelah terbentuk file .exe, gunakan perintah
```bash
./source
```
jika ingin mengubah masukkan, ubah file `input.txt`
output dapat dilihat pada terminal dan file `output.txt`

## Menjalankan File Pada Bin
pada /bin gunakan perintah
```bash
./source
```

jika ingin mengubah masukkan, ubah file `input.txt`
output dapat dilihat pada terminal dan file `output.txt`
